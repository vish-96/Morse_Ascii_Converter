# Morse_Ascii_Conveter

In order to use the converter. Use the executables in bin directory
Example:
  ./ascii_morse VISHAL_MORSE_CONVERTER
  OUTPUT:
    ...-..........-.-..---------.-.....-----.-.----....-..-.-..-.
  ./morse_ascii ...-_.._..._...._.-_.-.._----_--_---_.-._..._._----_-.-._---_-._...-_._.-._-_._.-.
  OUTPUT:
  VISHAL MORSE CONVERTER

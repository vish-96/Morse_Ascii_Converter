# Morse_Ascii_Converter

Example:<br />
  ./ascii_morse VISHAL_MORSE_CONVERTER<br />

    ...-..........-.-..---------.-.....-----.-.----....-..-.-..-.<br />
  ./morse_ascii ...-_.._..._...._.-_.-.._----_--_---_.-._..._._----_-.-._---_-._...-_._.-._-_._.-.<br />
  OUTPUT:<br />
  VISHAL MORSE CONVERTER<br />
